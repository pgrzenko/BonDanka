# BonDanka

Prosta porównywarka obligacji skarbowych COI (4-letnie) i EDO (10-letnie) indeksowanych inflacją — stworzona z myślą o osobach, które nie lubią Excela.

## Co robi aplikacja?

BonDanka w 4 prostych krokach pomaga odpowiedzieć na pytanie: **„Co lepiej wybrać — COI czy EDO w mojej sytuacji?"**

1. **Horyzont** — na ile lat chcesz zainwestować
2. **Kwota** — ile chcesz zainwestować
3. **Inflacja** — jakiej inflacji się spodziewasz
4. **Konto oszczędnościowe** — oprocentowanie konta, na które trafiają odsetki COI

Na końcu dostajesz jasną rekomendację, porównanie zysków netto (po podatku Belki) i wykres.

## Technologie

- React + TypeScript
- Vite
- Tailwind CSS v4
- Recharts (wykres porównawczy)

## Uruchomienie lokalne

```bash
npm install
npm run dev
```
